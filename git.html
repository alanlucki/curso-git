Comando Descripcion
git init inicializa el repositorio
git add [ nombre del archivo ] agrega los cambios de determinado archivo a stage
git commit registra cambios que actualmente se encuentran en el stage
git log lista todos los commits
git log --oneline lista todos los commits resumidos en una linea cada uno
git log --oneline | cat lista todos los commits resumidos en una linea cada uno, sólo con su descripción
git log --oneline --decorate lista todos los commits resumidos en una linea cada uno, con su punteros
git log --oneline --graph lista todos los commits resumidos en una linea cada uno, en un grafo
git log --oneline --all --graph --decorate lista todos los commits resumidos en una linea cada uno, en un grafo y sus b punteros
git diff [ nombre del archivo ] muestra los cambios de determinado archivo
git commit --amend agrega los cambios al commit
git checkout [ nombre del archivo ] revierte todos los cambios en determinado archivos
git reset [ nombre del archivo ] revierte todos los que se eencuentran en el stage en determinado archivos
git reset [ hash del commit ] revierte el repositorio al commit indicado como parámetro
git reset hard [ hash del commit ] restaura el repositorio sin guardar los cambios en el stage
git reset hard [ hash del commit ] restaura el repositorio sin guardar los cambios en el stage
git reset soft [ hash del commit ] restaura el repositorio y guarda los cambios en el stage
git diff [ hash del commit ] [ hash del commit ] muestra la diferencia entre 02 commits
git diff HEAD~1 HEAD muestra la diferencia entre el commit donde apunta 1 antes al HEAD contra el HEAD
git revert [ hash del commit ] revierte las acciones realizadas en el commit indicado, con acciones exactamente opuestas
git revert --no-commit [ hash del commit ] revierte las acciones realizadas en el commit indicado, con acciones exactamente opuestas, sin registrar un nuevo commit
git branch lista las ramas
git branch [nombre de rama nueva] crea una nueva rama a partir de la rama en la que actualmente nos ubicamos
git checkout [nombre de rama] cambia a la rama [nombre de rama]
git checkout -b [nombre de rama nueva] crea la rama [nombre de rama nueva] y nos ubica en dicha rama
git branch -m [nombre antiguo de rama] [nombre nuevo de rama] cambia el nombre de la rama [nombre antiguo de rama] por [nombre nuevo de rama]
git branch -d [nombre de rama] elimina rama [nombre de rama]
git branch -h lista los comandos que se pueden utilizar con branch
